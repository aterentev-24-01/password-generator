# О проекте 
Генератор случайных паролей
Простой скрипт для создания случайных паролей.
Если кому-то сложно самому придумывать пароль, то этот скрипт поможет создать надежный пароль желаемой длины. 

# Функциональность 
- Генирирует случайный пароль
- Количество символов можно выбрать

# Требования
- Python 3.6+

# Установка
git clone https://github.com/your-username/password-generator.git
создания виртуального окружения- bash
venv\Scripts\activate

# Запуск
cd src
python password_generator.py
